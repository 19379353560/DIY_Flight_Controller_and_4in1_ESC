# Contributing

Thanks for reviewing the 75A 4-in-1 ESC project. The best contributions are
specific hardware-review notes before anyone orders or powers boards.

## Helpful Feedback

- High-current copper paths, bottlenecks, vias, and connector assumptions.
- MOSFET placement, gate-driver routing, and switching-loop concerns.
- Thermal assumptions for a 75A x4 target.
- DSHOT signal integrity and manufacturing/assembly risks.
- References to comparable ESC designs or layout examples.

## Safety

This is high-current hardware. Do not treat repository files as production-ready
without independent electrical, thermal, and manufacturing review.
